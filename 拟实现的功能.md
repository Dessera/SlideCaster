# 拟实现的功能

## Essential

- 训练工具

    - 数据采集工具
    - 模型训练工具

- 便于模型集成的识别模块

    - sc_utils 会引用本地路径作为模型位置，在前端页面请求修改配置时，会返回一份当前的配置结构体（和元数据？）前端根据配置结构生成（匹配）配置页面，达成动态配置。
    - 用户需要实现的功能：
        
        1. listener(): Generator
        2. get_config(): Config
        3. set_config(cfg: Config)
        4. refresh()

- 服务器文件管理

    - 上传、下载文件、文件列表
